INNOVATION FROM SCRAP

Technical Projects

100% DIY

01. Custom Remote Controlled Robot

Objective: To design a low-cost RC robot using molded PVC and salvaged electronics.

🧱 Build & Materials

  • Chassis: Heated & molded PVC pipe sheets
  • Eyes/Nose: Bicycle reflectors & damaged LEDs
  • Cooling: Fan from a bubble-blowing toy
  • Control: RF module from a ₹150 toy car

⚙️ Working Principle

RF signals trigger the motor driver via a salvaged receiver. Features a speed regulator from an AC fan and a handmade moving-hand mechanism.

Recycled

02. Custom Wireless Headphones

Objective: Creating high-utility audio gear from damaged neckbands and mobile parts.

🧱 Build & Materials

  • Frame: PVC pipe (cut, heated, and bent)
  • Audio: Bluetooth module from a broken neckband
  • Speakers: Salvaged from keypad mobile phones
  • Mic: Detachable mobile microphone

⚙️ Working Principle

Pairs via Bluetooth to receive wireless signals. Power is managed by a salvaged Li-ion battery circuit from the original neckband.

Exploratory

03. Advanced Saline Monitoring System

Insight: This project served as an exploration into multi-sensor data fusion using ESP32.

🔩 Actual Components

  • Core: ESP32 Microcontroller
  • Environment: DHT11 Temp/Humidity Sensor
  • Medical: Water Flow Sensor & BPM Heart Rate Sensor
  • Setup: Breadboard Prototype

⚙️ Working Principle

The ESP32 processes real-time data from the BPM and Flow sensors to track patient vitals and saline depletion, displaying data for clinical review.
